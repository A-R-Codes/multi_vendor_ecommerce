## Multi Vendor Ecommerce System (Laravel)

A full featured multi vendor ecommerce marketplace built with Laravel, MySQLi, and integrated payment gateway support. This platform allows multiple vendors to register, manage their own stores, and sell products through a single centralized system with secure checkout and admin supervision.

The system is designed to handle real world ecommerce operations including vendor management, product listings, order processing, and online payments.

---

### Main Features

#### Admin Panel

* Manage vendors and approve registrations
* Manage categories, products, and users
* View and control all orders
* Commission management system
* Sales and performance reports
* Platform level settings control

#### Vendor Panel

* Vendor registration and secure login
* Add, edit, and delete products
* Inventory and stock management
* Order management
* Earnings tracking
* Sales analytics and reports

#### Customer Panel

* User registration and authentication
* Browse products by category and vendor
* Add to cart and wishlist
* Secure checkout process
* Online payment gateway integration
* Order tracking and order history

---

### Core Functionalities

* Multi vendor marketplace architecture
* Role based authentication (Admin, Vendor, Customer)
* Dynamic product and category management
* Shopping cart system
* Complete order lifecycle management
* Secure payment gateway integration
* Responsive user interface
* Secure database handling using MySQLi

### Technologies Used

* Laravel (Backend Framework)
* PHP

---

### Project Purpose

This project demonstrates a scalable ecommerce solution where multiple sellers can operate independently within one platform while the admin maintains full control over the marketplace operations, commissions, and system management.

Suitable for learning, portfolio demonstration, and real world ecommerce deployment with further customization.
